ESTOS SON LOS PASOS PARA CREAR UN REPOSITORIO EN GITHUB

Publicar un sitio web en Github es un poco facil. Primero se hace una cuenta y se crea un repositorio nuevo. Despues se suben los archivosa la pagina, como el archivo principal "index.html", y tambien las imagenes o estilos que tenemos. Es muy importante que el archivo principal se llame "index.html" para que la pagina funcione bien. Luego vas a SETTINGS, se busca la opcion PAGES y se elige la rama MAIN para activar el sitio. Se guardan los cambios y en unos minutos Git hub nos dara un enlace. Ese enlace es la pagina en internet y se puede compartir.
